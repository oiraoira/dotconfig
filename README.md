# Dot Files Config

This repo contains my basic linux dot files to ease migration between computers

 * .bashrc
 * .vimrc 
 * .vim

## Usage

On a fresh system,

* install git and ssh
* clone this repo

```bash
cd dotconfig && ./install.sh
```

* on debian, add exuberant tags to vim

```bash
sudo apt install exuberant-ctags
```
