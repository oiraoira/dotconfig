# Dot Files Config

This repo contains basiclinux system dot files to ease migration.

 * .bashrc
 * .vimrc and .vim

## Usage

On a fresh system,

* ssh keys
* clone this repo
* checkout submodules
  - git submodules init
  - git submodules update
* soft link .bashrc and .vim
* add third-party package for vim
  - sudo apt install exuberant-ctags
